微觀經濟、宏觀經濟 - 理解經濟運作

經濟學知識重點歸納

供需理論

  • 需求法則：價格上升，需求量減少
  • 供給法則：價格上升，供給量增加
  • 市場均衡：供需相等時的價格和數量
  • 彈性分析：價格彈性、所得彈性、交叉彈性

市場結構

  • 完全競爭：價格接受者，長期均衡P=MC=AC
  • 獨占：單一賣者，MR=MC定價
  • 寡占：少數賣者，相互依存
  • 獨占性競爭：產品差異化，短期有超額利潤

消費者理論

  • 效用理論：邊際效用遞減法則
  • 無異曲線：相同效用水準的組合
  • 預算線：消費者負擔得起的組合
  • 消費者均衡：MRS = Px/Py

生產者理論

  • 生產函數：投入與產出的關係
  • 邊際產量：增加一單位投入的產量
  • 成本分析：固定成本、變動成本、邊際成本
  • 利潤最大化：MR = MC

國民所得

  • GDP：國內生產毛額
  • GNP：國民生產毛額
  • 計算方法：支出法、所得法、生產法
  • 實質GDP：消除物價變動影響

通貨膨脹

  • CPI：消費者物價指數
  • 通膨率：(今年CPI-去年CPI)/去年CPI×100%
  • 通膨類型：需求拉動、成本推動、結構性
  • 通膨影響：所得重分配、資源配置扭曲

失業問題

  • 失業率：失業人口/勞動力×100%
  • 失業類型：摩擦性、結構性、循環性
  • 自然失業率：摩擦性+結構性失業
  • 菲力普曲線：通膨與失業的取捨關係

貨幣政策

  • 貨幣供給：M1A、M1B、M2
  • 貨幣政策工具：重貼現率、存款準備率、公開市場操作
  • 貨幣數量學說：MV = PY
  • 利率理論：流動性偏好理論

國際貿易

  • 比較利益：機會成本較低的商品
  • 絕對利益：生產效率較高的商品
  • 貿易條件：出口價格/進口價格
  • 匯率制度：固定匯率、浮動匯率

經濟成長

  • 成長率：(今年GDP-去年GDP)/去年GDP×100%
  • 成長因素：勞動、資本、技術進步
  • 索羅模型：長期成長決定於技術進步
  • 內生成長：人力資本、研發投資

詳細章節學習

第一章：基本經濟概念

1.1 經濟學作為社會科學

定義：經濟學是研究在資源稀少的情況下，個人與社會如何作出選擇以最大化利益分配的社會科學。
  • 核心假設：人類行為基於有限資源與無窮慾望的衝突
  • 研究範疇：個體經濟學（微觀）與總體經濟學（宏觀）
  • 方法論：實證分析與規範分析的區別

1.2 稀少性、選擇與機會成本

機會成本公式：機會成本 = 所放棄的最佳選項的價值
  • 稀少性：相對於人類的無限慾望，資源供應是有限的
  • 免費物品 vs 經濟物品：供應量與需求量的比較
  • 顯性成本與隱性成本：實際支付 vs 放棄的潛在收益
  • 沉沒成本：過往已付出且無法收回的資源

1.3 三大基本經濟問題

生產什麼？
社會應該生產哪些商品和服務
如何生產？
採用何種生產技術和要素組合
為誰生產？
商品和服務如何在社會成員間分配

1.4 專門化與交易

  • 相互依存：交易是專門化的必要條件
  • 經濟循環：生產 → 收入 → 支出 → 生產
  • 實證性 vs 規範性：「是什麼」vs「應該是什麼」

第二章：廠商、企業與生產

2.1 廠商定義與所有權形式

公有制企業
  • 國有企業：政府全資擁有
  • 集體企業：集體所有制
私人制企業
  • 獨資企業：單一擁有者
  • 合夥企業：多個合夥人
  • 有限公司：股東有限責任

2.2 生產種類與階段

初級生產（第一產業）

農業、漁業、林業、礦業

二級生產（第二產業）

製造業、建築業、公用事業

三級生產（第三產業）

金融、貿易、運輸、教育、醫療

2.3 生產要素

勞動力
  • 勞力供應：數量與質量
  • 生產力：單位勞動投入的產出
  • 流動性：職業與地區流動
企業家才能
  • 風險承擔：不確定性管理
  • 決策職能：戰略規劃
  • 組織協調：團隊建設
天然資源
  • 不可改變性：位置固定
  • 可再生 vs 不可再生
  • 土地資源的多樣性
人造資源（資本）
  • 資本累積：投資過程
  • 折舊：物理、技術、經濟
  • 利息：資本成本

第三章：市場、需求與價格

3.1 廠商目標

利潤極大化條件：邊際收入(MR) = 邊際成本(MC)
  • 利潤極大化：總收入與總成本差額的最大化
  • 市場佔有率最大化：低價競爭、產品差異化
  • 企業社會責任：環境保護、員工福利、社區貢獻

3.2 市場需求

需求定律：在其他條件不變下，商品價格與需求量呈反向關係
需求變化因素：
  • 消費者收入
  • 消費者偏好
  • 相關商品價格
  • 預期價格
商品類型：
  • 優等物品（正常品）
  • 劣等物品
  • 替代品
  • 輔助品（互補品）

3.3 市場供應

供應定律：在其他條件不變下，商品價格與供應量呈正向關係
  • 供應變化因素：生產成本、技術、政策、預期價格
  • 聯合供應：生產一種商品時同時產出另一種商品
  • 競爭性供應：相同資源可用於生產不同商品

3.4 均衡價格與數量

均衡條件：需求量(Qd) = 供應量(Qs)
市場調節機制：
  • 價格高於均衡：供過於求，價格下降
  • 價格低於均衡：供不應求，價格上升
盈餘分析：
  • 消費者盈餘：願付價格 - 實際價格
  • 生產者盈餘：實際價格 - 最低接受價格

3.5 價格彈性

需求價格彈性：Ed = (ΔQ/Q) ÷ (ΔP/P)
彈性分類：
  • 完全彈性：Ed = ∞
  • 彈性：Ed > 1
  • 單位彈性：Ed = 1
  • 缺乏彈性：0 < Ed < 1
  • 完全缺乏彈性：Ed = 0
影響因素：
  • 替代品的可得性
  • 商品的必需性
  • 支出比重
  • 調整時間

第四章：競爭、市場結構與效率

4.1 市場結構

完全競爭
  • 眾多買賣雙方
  • 產品同質
  • 信息完全
  • 自由進出
  • 價格接受者
壟斷性競爭
  • 廠商數量較多
  • 產品差異化
  • 進入相對容易
  • 有限市場力量
寡頭壟斷
  • 廠商數量少
  • 相互依賴
  • 進入障礙高
  • 可能串謀
壟斷
  • 單一賣方
  • 無近似替代品
  • 進入障礙極高
  • 價格制定者

4.2 效率、公平與政府角色

經濟效率條件：
  • 配置效率：MSB = MSC
  • 生產效率：最低成本生產
  • 動態效率：促進創新
市場失效原因：
  • 價格干預
  • 稅項與津貼
  • 配額限制
  • 壟斷力量
  • 外部性問題

4.3 政府干預措施

價格上限
  • 目的：保護消費者
  • 影響：供不應求
  • 實例：租金管制
價格下限
  • 目的：保護生產者
  • 影響：供過於求
  • 實例：最低工資
從量稅
  • 影響：價格上升，數量減少
  • 稅負分配：彈性小的一方承擔更多
  • 效率影響：產生淨損失
津貼
  • 影響：價格下降，數量增加
  • 政策目的：鼓勵消費或生產
  • 社會效益：支持特定行業
